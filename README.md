Symfony Eclipse Plugin
======================

[![Build Status](https://secure.travis-ci.org/pulse00/Symfony-2-Eclipse-Plugin.png)](http://travis-ci.org/pulse00/Symfony-2-Eclipse-Plugin)

- Java 6
- Eclipse 4.4 (Luna)

### Recommended

- Java 7
- Eclipse 4.5 (Mars)
