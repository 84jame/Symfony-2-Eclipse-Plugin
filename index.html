What's this good for?

The Symfony Eclipse Plugin provide support for the Symfony2 PHP Framework. It is built on top of the Dynamic Languages Toolkit and the PHP Development Tools. and adds some useful features which hopefully save you some time like:

  • - Codeassist for elements specific to the Symfony2 framework, like services, routes and template paths
  • - Annotation support
  • - Twig support

Disclaimer

The plugin is still beta. Please report problems and feature requests via github issues.

Prerequisites

At the current stage the Symfony2 Eclipse Plugin require a nightly build of the PHP Development Tools (PDT), as there have been some changes to the way extenders can hook into PDT. This nightly build can be downloaded as a build artifact from the eclipse continuous integration site. You'll need to download a file called pdt-Update-N201107010913.zip

All following installation/deinstallation steps can be found via Help -> Install new software...

Before installing the nightly build, make sure to uninstall any existing installations of the PDT plugin, if one exists.

After that, open the "Install new software" dialog and add a new repository. Select "Local" and browse to the unzipped PDT nightly build. The repository will offer a couple of PHP features, choose the "All-In-One SDK" Feature and install it.

This installation step well be redundant as soon as the next maintenance release of PDT is deployed.

Installation via Updatesite

The plugin can be installed via the regular update site mechanism.

The update site is located at

http://pulse00.github.com/Symfony-2-Eclipse-Plugin/


After adding the updatesite to your eclipse installation, you will see 2 Symfony2 features:

  • - Symfony2 Core Feature
  • - Symfony2 Twig Support
The Core Feature is mandatory. The Twig Support Feature is only needed if you use the Twig templating engine. If you need Twig Support, install the separate Twig plugin first.

Usage

To make your existing PHP Projects Symfony2 aware, add the SymfonyNature by right-clicking on the project in the PHP explorer -> Symfony -> Add/Remove Symfony Nature. This will add the Symfony2 project nature and trigger the indexing mechanism of DLTK/PDT/Symfony (see the status bar at the bottom).

That's it, you're ready to go.
