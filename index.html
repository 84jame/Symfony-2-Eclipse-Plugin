Symfony 2 - Eclipse Plugin

An extension to the PDT plugin of eclipse to support Symfony 2 development

The plugin is still under development. We target a release date together with the final release of Symfony 2. Please feel free to add feature wishes to the issue tracker or vote for an existing issue.

We can always need help in developing the plugin, so if you want to join the team, please drop me a message
